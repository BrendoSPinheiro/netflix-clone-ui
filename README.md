### ⚙ How to use

**Cloning repository and running**

```bash
$ git clone https://github.com/BrendoSPinheiro/netflix-clone-ui && cd netflix-clone-ui
```

**Installing dependencies**

```bash
$ yarn
```

**Getting Started**

```bash
$ yarn start
```
